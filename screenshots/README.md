When you run LlamaFactory locally, capture these screenshots and store them here:

- training_config.png — show the LoRA/optimizer settings
- training_loss.png — loss curve from the run
- inference_baseline.png — baseline inference on held-out examples
- inference_finetuned.png — finetuned model outputs

Use full-resolution PNGs. Filenames above are referenced in `report.md`.
